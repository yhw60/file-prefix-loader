# file-loader

A file loader module for webpack

## Getting Started
To begin, you'll need to install `file-prefix-loader`:

```console
$ npm install file-prefix-loader --save-dev
```

## Options
```js
// webpack.config.js
...
{
  loader: 'file-prefix-loader',
  options: {
    prefix:'/a'
  }
}
...
```
